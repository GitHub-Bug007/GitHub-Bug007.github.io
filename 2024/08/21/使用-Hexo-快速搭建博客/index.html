官方文档

  • Hexo

所需环境：

  • Node.js(Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

本文章使用的环境版本
node.js v14.18.2
git v2.34.1.windows.1
hexo v7.3.0
hexo-cli v4.3.2

为什么用 Hexo

速度 快、简洁 且 高效，插件多，使用 MarkDown 语法

搭建博客

在所需目录使用 git 打开

解释：

  • <folder>：替换成自己所需的文件名
  • hexo init <folder>：初始化博客文件夹
  • cd <folder>：进入博客文件夹
  • npm install：加载所需的依赖

博客文件夹目录

  • 详细说明 解释：
  • node_modules：插件目录
  • scaffolds：模板文件夹
  • source：资源文件夹
  • _config.yml：站点配置文件
  • package.json：引用程序的信息
  • themes：主题文件夹

  • hexo deploy：推送到 github 等仓库（需要配置 deploy）
